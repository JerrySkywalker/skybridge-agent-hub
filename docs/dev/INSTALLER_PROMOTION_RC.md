# Installer Promotion RC

`v1.9.0-installer-promotion-rc` is the next sandbox-only installer/product release-candidate layer.

It adds:

- installer artifact promotion gate;
- artifact provenance;
- release artifact manifest and checksum validation;
- long bounded sandbox soak;
- update channel manifest preview;
- offline update and rollback preview;
- host mutation permission model preview;
- installer safety interlock;
- operator acceptance v4;
- read-only Web/Desktop release candidate surfaces.

No real installer is run. No host mutation, network update, manual upload, or manual GitHub Release creation is allowed.
