# Productization Next Roadmap

Recommended next goals after the local productization RC:
- installer signing plan;
- authenticated local pairing flow;
- operator-controlled service install preview;
- UI report ingestion from `.agent/tmp` metadata;
- release artifact promotion policy.

Execution, workunit apply, task claim, queue apply, remote execution, and arbitrary command dispatch remain disabled until a future goal explicitly authorizes and gates them.
