# Installer Soak RC

The v1.8 installer soak RC is a local productization release candidate for the sandboxed installer path.

It includes:

- release workflow side-effect guard
- tag publish policy
- sandboxed installer candidate
- installer manifest and verification
- sandbox-installed runtime rehearsal
- install, upgrade, rollback soak
- crash/recovery sandbox
- operator acceptance v3
- read-only Web/Desktop acceptance surfaces

The RC remains sandbox-only and does not install to the host.
