# Naming

Long-term product name: **SkyBridge Agent Hub**.

Rationale:

- not locked to Codex;
- not locked to Glance;
- not only a dashboard;
- not only a thesis-defense YOLO helper;
- emphasizes the bridge between local agents, cloud services, notifications and future remote-control apps.

Canonical identifiers:

```text
repo/folder: skybridge-agent-hub
package scope: @skybridge-agent-hub/*
event schema: skybridge.agent_event.v1
server service: skybridge-server
sidecar service: skybridge-sidecar
deploy path: /opt/skybridge-agent-hub
```
